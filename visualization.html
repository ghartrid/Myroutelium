The Key Insight: Biological SDN

Real fungi use two signaling systems at different speeds — just like software-defined networking separates control and data planes. Calcium signals (control plane) travel orders of magnitude faster than nutrient transport (data plane), giving the network instant awareness of state changes before the slower reinforcement learning adapts.

How It Works

Each character is converted to 8-bit ASCII. The source electrode injects a current pulse for each 1 bit and stays silent for each 0 bit. The resulting action potentials propagate through the mycelial network as Ca2+ waves along hyphal segments. The destination electrode detects spikes and reconstructs the binary stream.

Signal quality depends on the biological path — thicker rhizomorphs carry signals further with less attenuation. This is a living, biological data link.

Mycelium Encryption — Living Cryptography

Four security layers:
1. Secret Sharing — Data is split into N shares across N biological paths. No single path carries the complete message (Shamir-like).
2. Biological Path Signature — Each share is XOR-masked with a key derived from the path's physical properties (segment diameters, lengths, ion channel states). The network topology is the encryption key.
3. Temporal Key Rotation — The mycelium grows and changes continuously. The key is different every time — a living one-time pad.
4. Physical Security — Tapping a single hypha yields only a masked fragment. Destroying the network destroys the key. No digital copy of the key ever exists.
